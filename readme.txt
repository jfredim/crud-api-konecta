Para la instalacion del crud se deben tener en cuenta:

- ejecutar dtabase.sql : crea la base de datos y la tabla productos

- dbconfig.php: administra base de datos

- index.php: muestra los registros existentes en la base de datos
- create.php: crea los registros en la base de datos
- update.php: actualiza registros de la base de datos
- delete.php: elimina registros de la base de datos
- busqueda.php: busca registros de la base de datos

- venta.php: resta unidades del stock de un producto

**** Todos los programas que tienen movimientos fueron probados y ejecutados por POSTMAN para su validacion y ejecucion
